# stud_vibe_coding

個人網站專案 - 蔡耀德醫師（胖胖醫師）

## 專案簡介

這是一個現代化的個人網站，展示醫師的專業背景、學經歷、專長技能與聯絡方式。

## 檔案結構

```
├── index.html          # 主要 HTML 檔案
├── style.css           # 樣式表檔案
├── script.js           # JavaScript 互動功能
└── README.md           # 專案說明文件
```

## 功能特色

- 📱 響應式設計，支援各種螢幕尺寸
- 🌓 深色/淺色主題切換
- ✨ 流暢的動畫效果
- 📧 聯絡方式連結
- 🎨 現代化的使用者介面

## 技術架構

- **HTML5** - 網頁結構
- **CSS3** - 樣式與動畫
- **JavaScript (ES6+)** - 互動功能
- **Font Awesome** - 圖示
- **Google Fonts** - Noto Sans TC 字型

## 使用方式

1. 直接開啟 `index.html` 即可在瀏覽器中檢視
2. 或使用任何靜態網站伺服器託管

## 網站內容區塊

- **關於我** - 個人簡介與專業介紹
- **學經歷** - 教育背景與工作經歷
- **專長與技能** - 醫療專業領域
- **榮譽獎項** - 專業認證與獎項
- **代表課程** - 教學與講座主題
- **聯絡方式** - 社群媒體與電子郵件連結
